    Discover Devices

    Default: only devices with groupId=0 (Unconfigured) reply. Use "Discover in" to re-poll a configured group, or "All groups" to sweep every known group sequentially.

    Re-sync group config

    Re-broadcasts every device's stored groupId to the network so the radio state matches the host's view. Online devices ACK in ~300 ms; offline devices time out at ~4.7 s per node unless skipped.

    Skipped devices auto-resync on their next IDENTIFY/STATUS reply when they come back online. Leave unchecked to push to every known device now.

    Firmware Update

    Activates the AP for each node sequentially via RaceLink, connects to the node AP using nmcli (NetworkManager), and checks the full MAC via /json/info before performing the OTA update.

    Default AP-IP: 4.3.2.1
    Host scans for any of the SSIDs and connects with this password (WLED's default is "wled1234")
